How to make a webpage

Making a web page is a very easy process. It includes using programing languages:

HTML( Hypertext Markup Language)

It is used to make the basic structure of a website. It is refered to as a declarative language since all actions must be declared ((made specific)). HTML is considered one the easiest programing languages and is widely used to make websites and webpages for different brands and companies.

CSS

It is is an advanced programing language that is used to make a website more interactive. It creates buttons and links in which the user can click at to make an action to take place. programmers use the language by enaling it to obey commands when a certain action takes place.
PYTHON
It is a very advancer programing language that enables a proggramer to make their website more attractive and appealing. It involves use of graphics to make a website more appealing. Websites can be used for the following resons:
  • Advertising a company's product
